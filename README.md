## This is an example of using Laravel Reverb websocket with several examples.

### Install
```shell
composer install
```

### NPM Install
```shell
npm install
```

### Make .env & Configure
```shell
cp .env.example .env
```

### Generate key
```shell
php artisan key:generate
```

### Migrate
```shell
php artisan migrate --seed
```

### NPM Run
```shell
npm run dev
```

### Start Reverb
```shell
php artisan reverb:start
```
